Crop Disease Prediction

The Crop Disease Prediction System is an AI-powered web application that helps farmers and agricultural experts detect crop diseases early. Using deep learning and computer vision, it analyses leaf images to identify diseases across crops like cashew, cassava, guava, maize, pumpkin, and tomato. Built with Flask, it offers a user-friendly interface for image uploads, instant predictions, and treatment recommendations. The system uses a CNN model for accurate classification, along with user authentication, secure image handling, and prediction history storage. It aims to promote precision farming, reduce crop losses, and support sustainable agriculture.

Client Relationship Management

This web-based Client Relationship Management (CRM) system, built using the MERN stack, manages projects through five roles: Admin, Client, Project Manager, Team Leader, and Team Member. It allows clients to submit projects, track progress, and give feedback. Project Managers approve projects, assign tasks, and oversee progress. Team Leaders manage team members and update project status, while Team Members handle assigned tasks and communicate any issues. The system ensures smooth collaboration and efficient project delivery.

Employee Record Management

This system is designed to maintain company employee information in a database managed by an administrator. The administrator can perform CRUD operations on employee records, filter them, and search for specific employees using their ID. Form validation ensures accurate data entry and that employee details can be import with a CSV file. The proposed model was developed using MEAN stack Technology.
